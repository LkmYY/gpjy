# 股票分析软件

这是一个基于Python的股票分析软件，提供以下功能：

- 股票数据获取和分析
- 技术指标计算
- 新闻分析
- 可视化界面

## 安装

1. 克隆项目
2. 安装依赖：
```bash
pip install -r requirements.txt
```

## 使用方法

运行主程序：
```bash
streamlit run app.py
```

## 项目结构

- `app.py`: 主程序入口
- `data/`: 数据获取和处理模块
- `analysis/`: 技术分析模块
- `news/`: 新闻分析模块
- `utils/`: 工具函数
